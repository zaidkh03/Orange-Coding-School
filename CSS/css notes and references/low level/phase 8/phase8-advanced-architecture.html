1. CSS Custom Properties (Variables)

CSS variables are reusable values defined once and used everywhere. They are declared using --variable-name and accessed with var().

2. Native CSS Nesting

Native nesting allows writing CSS rules inside other rules. This makes component relationships easier to read.

3. Logical Properties

Logical properties replace left/right/top/bottom with start/end/block/inline directions. This automatically adapts layouts for RTL languages.

4. Cascade Layers (@layer)

Cascade layers control CSS priority intentionally. Instead of relying on load order or selector specificity, layers explicitly define rule hierarchy.
